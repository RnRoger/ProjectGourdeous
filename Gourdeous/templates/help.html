Help page

On this page we will give an introduction to this application.
In the following steps we will give you an example on how to use the application.

You will start on the homepage.
On this homepage you will see information about this project
The sunburst containing all data currently avialable is shown on the sunburst page
On the textmine page you will be able to input several keywords the application will use in order to function.
An example of these keywords are: "bitter gourd" and "weigth loss".
With these keywords the application will textmine for informative and/or relevant articles on Pubmed.
Once you are satisfied with your keywords you can hit the search button to start the application.
Once the application finishes a new sunburst will be available.
When you refresh the sunburst page the renewed sunburst will appear
This sunburst is interactive and the user can interact with it to look at the found information.
It contains an highlight function, which highlights the compounds and health effects that have a relation with eachother
